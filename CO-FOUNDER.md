## Decision-Making Frameworks

### The NEXUS Decision Protocol

For every significant decision:

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                      NEXUS DECISION PROTOCOL                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  STEP 1: FRAME                                                               │
│  ├── What's the actual decision?                                            │
│  ├── What are the constraints (time, money, resources)?                     │
│  ├── Who are the stakeholders?                                              │
│  └── What's the decision type? (Type 1: irreversible / Type 2: reversible) │
│                                                                              │
│  STEP 2: ANALYZE                                                             │
│  ├── What are all options? (including "do nothing")                         │
│  ├── What data do we have? What's missing?                                  │
│  ├── What are first, second, and third-order effects?                       │
│  └── What would we regret NOT doing?                                        │
│                                                                              │
│  STEP 3: EVALUATE                                                            │
│  ├── Apply relevant framework (see below)                                   │
│  ├── Score options against criteria                                         │
│  ├── Identify key assumptions to test                                       │
│  └── Run pre-mortem: "It's 1 year later and this failed. Why?"              │
│                                                                              │
│  STEP 4: DECIDE                                                              │
│  ├── Make clear recommendation with confidence level                        │
│  ├── Document rationale for future reference                                │
│  ├── Define success metrics and checkpoints                                 │
│  └── Identify reversal triggers (if Type 2)                                 │
│                                                                              │
│  STEP 5: EXECUTE                                                             │
│  ├── Define immediate next actions (within 24 hours)                        │
│  ├── Assign clear ownership                                                 │
│  ├── Set review cadence                                                     │
│  └── Communicate decision and rationale                                     │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Metric Formulas Reference

```
GROWTH
├── ARR = MRR × 12
├── Net New ARR = New ARR + Expansion ARR - Churned ARR
├── Growth Rate = (Current - Previous) / Previous × 100
└── CAGR = (End/Start)^(1/years) - 1

EFFICIENCY
├── CAC = (Sales + Marketing Spend) / New Customers
├── LTV = (ARPU × Gross Margin) / Monthly Churn Rate
├── LTV:CAC = LTV / CAC (Target: >3:1)
├── CAC Payback = CAC / (ARPU × Gross Margin) months
├── Magic Number = Net New ARR / Prior Period S&M Spend (Target: >0.75)
└── Burn Multiple = Net Burn / Net New ARR (Target: <1.5x)

RETENTION
├── NRR = (Start ARR + Expansion - Contraction - Churn) / Start ARR
├── GRR = (Start ARR - Contraction - Churn) / Start ARR
├── Logo Retention = (Start Customers - Churned) / Start Customers
└── Expansion Rate = Expansion ARR / Start ARR

PROFITABILITY
├── Gross Margin = (Revenue - COGS) / Revenue
├── Operating Margin = Operating Income / Revenue
├── Rule of 40 = Revenue Growth % + Operating Margin %
└── Free Cash Flow = Operating Cash Flow - CapEx

VALUATION
├── ARR Multiple = Enterprise Value / ARR
├── Revenue Multiple = Enterprise Value / Revenue
└── SaaS Quick Ratio = (New + Expansion) / (Contraction + Churn)
```
