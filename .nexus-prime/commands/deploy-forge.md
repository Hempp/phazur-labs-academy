# Deploy Software Engineer Master Team

Activating the FORGE-X engineering team with specialized agents for comprehensive code work.

## Deployment Modes

**Full Team (default):**
```
/deploy-forge
```

**Audit-Focused:**
```
/deploy-forge audit
```

**Security-Focused:**
```
/deploy-forge security
```

**Performance-Focused:**
```
/deploy-forge performance
```

## Team Composition

| Agent | Role | Specialty |

| **FORGE-X** | Lead Architect | Strategy, code quality, synthesis |

## Quick Start

1. **Audit a codebase:**
   ```
   /deploy-forge
   /forge-audit ./src
   ```

2. **Run security scan:**
   ```
   /forge-security ./src
   ```

3. **Profile performance:**
   ```
   /forge-perf ./api
   ```

4. **Generate tests:**
   ```
   /forge-test-gen ./src/utils
   ```

## Available Commands

### Audits
- `/forge-audit [target]` - Comprehensive code audit
- `/forge-audit-quick [target]` - 5-minute scan
- `/forge-audit-security [target]` - Security focus
- `/forge-audit-performance [target]` - Performance focus

### Code Operations
- `/forge-review [PR/file]` - Code review
- `/forge-refactor [target]` - Intelligent refactoring
- `/forge-lint [target]` - Run linters
- `/forge-format [target]` - Format code

### Testing
- `/forge-test [target]` - Run tests
- `/forge-test-gen [target]` - Generate tests
- `/forge-test-coverage` - Coverage report
- `/forge-test-mutation` - Mutation testing

### Security
- `/forge-security [target]` - Security analysis
- `/forge-secrets-scan` - Detect leaked secrets
- `/forge-deps-audit` - Dependency vulnerabilities

### Performance
- `/forge-perf [target]` - Performance analysis
- `/forge-benchmark [function]` - Microbenchmark
- `/forge-profile [target]` - CPU/memory profiling

### Generation
- `/forge-scaffold [type]` - Project scaffolding
- `/forge-component [name]` - Component generation
- `/forge-api [spec]` - Generate API from spec
- `/forge-docs [target]` - Generate documentation

### Reference
- `/forge-pattern [name]` - Design pattern lookup
- `/forge-algo [algorithm]` - Algorithm reference
- `/forge-celebrate [achievement]` - Document wins
